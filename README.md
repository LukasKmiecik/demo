# Demo systemu serwisowego 3D — GitHub Pages

To jest statyczna wersja demo przygotowana specjalnie pod GitHub Pages.

## Jak uruchomić na GitHub Pages

1. Utwórz nowe repozytorium na GitHub.
2. Wrzuć do niego całą zawartość tego folderu.
3. Wejdź w **Settings → Pages**.
4. Wybierz gałąź `main` i folder `/root`.
5. Zapisz ustawienia.
6. Po chwili GitHub opublikuje stronę pod własnym adresem.

## Co zawiera demo

- statyczną stronę startową,
- prosty widok 3D w przeglądarce oparty o Three.js z CDN,
- panel listy komponentów,
- panel szczegółów i błędów,
- kartę komponentu siłownika przepustnicy,
- dane komponentów w pliku JSON.

## Następny krok

- dodanie dokumentacji PDF,
