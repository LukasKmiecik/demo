# Demo systemu serwisowego 3D - etap 1

To jest pierwsza, bardzo prosta wersja strony pod GitHub Pages:
- po lewej interaktywny model budynku GLB/GLB Draco,
- po prawej panel informacji o obiekcie.

## Struktura
- `index.html` - układ strony
- `style.css` - wygląd
- `js/app.js` - ładowanie modelu i obsługa widoku 3D
- `dane/obiekt.json` - informacje o budynku
- `assets/modele/budynek/willa_parkowa_10.glb` - model 3D

## Jak uruchomić na GitHub Pages
1. Wrzuć całą zawartość tego folderu do repozytorium.
2. W repo wejdź w `Settings -> Pages`.
3. Wybierz publikację z gałęzi `main` i folderu `/root`.
4. Po chwili strona będzie dostępna pod linkiem GitHub Pages.

## Następny krok
Po zatwierdzeniu tego etapu dokładamy model agregatu i tworzymy drugi poziom informacji.
